TSMIS Reports Exporter

Bulk-export Caltrans TSMIS reports for every California state route — from a single click.

Download the latest release View source on GitHub

TSMIS Reports Exporter is a free, open-source Windows desktop tool that automates exporting reports from the Caltrans Transportation System Management Information System (TSMIS). Instead of pulling reports one route at a time, you sign in once, choose the report types you want, and the tool walks every California state route for you. It ships as a single portable zip — unzip, double-click, done.
