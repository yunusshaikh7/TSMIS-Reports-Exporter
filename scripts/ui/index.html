What you'll get

A Summary (with self-check rows that prove the numbers line up), a cell-by-cell Comparison (matching values shown plainly, differences in red as “first value ≠ second value”), a Spot Check sheet to audit any single row, “Only in …” sheets collecting every one-sided row for each side, and the two inputs side by side — row numbers link between sheets throughout. All-routes comparisons also get a Routes sheet — which routes each side covers, what's missing where (whole-missing routes are flagged “entire route”), and how much differs per route. You'll be asked where to save it. For the reports that support evidence images (marked with a camera on the matrix pages), the vs TSN Matrix can also render sampled differences as highlighted snippets from both PDFs — its toggle lists exactly which reports qualify.

Reliability

Raise the time limits if the TSMIS server is slow and routes keep timing out. Changes apply to the next run.

Debugging

The support bundle is a zip of the logs, run reports and settings — everything a maintainer needs and never your login. “Revert” reinstalls the release before this one (your reports, login and settings are kept); it appears only on a writable installed copy.

TSMIS site addresses

The page each data source / environment lives at. Edit one if the site moves before an app update ships — it's used by the very next sign-in, export or Verify. Clear a box to go back to the built-in address (keep the ?env= and ?src= parameters unless the new site drops them).

Built-in Chromium

An unmanaged browser of the app's own (the one the “with-browser” download ships): org policy can't interfere with its sign-in window, and its version always matches this app. ~170 MB; it lives in the app's data folder and survives updates.

—
Export browser

Which browser does the actual exporting (plus fast-mode workers and the sign-in capture). Microsoft Edge is always used for one-click sign-in and as a fallback — this only chooses between Google Chrome and the app's Built-in Chromium, and only when both are installed.

TSN reports

The canonical home for each report's TSN source data (the comparison ground truth). Import the raw TSN file(s) once — district PDFs or a statewide workbook, depending on the report — then Rebuild turns them into the consolidated workbook every TSMIS-vs-TSN comparison reads. A green dot means the consolidated workbook is current; amber means it's missing or older than the raw you imported. After an app update the normalizer can move on, which marks reports out of date — Rebuild all out of date does them in one run.

Data & storage

“Delete all reports” clears every exported, consolidated and comparison file for a fresh start — logs, your login and these settings are always kept.
