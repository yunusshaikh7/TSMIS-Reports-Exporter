Pick one or both. With both, the values copy is saved next to the other as “… (values).xlsx”.

What you get

A Summary (with self-check rows that prove the numbers line up), a cell-by-cell Comparison (matching values shown plainly, differences in red as “first value ≠ second value”), a Spot Check sheet to audit any single row, “Only in …” sheets collecting every one-sided row for each side, and the two inputs side by side — row numbers link between sheets throughout. All-routes comparisons also get a Routes sheet — which routes each side covers, what's missing where (whole-missing routes are flagged “entire route”), and how much differs per route. You'll be asked where to save it.

Reliability

Raise the time limits if the TSMIS server is slow and routes keep timing out. Changes apply to the next run.

Debugging

The support bundle is a zip of the logs, run reports and settings — everything a maintainer needs and never your login.

TSMIS site addresses

The page each data source / environment lives at. Edit one if the site moves before an app update ships — it's used by the very next sign-in, export or Verify. Clear a box to go back to the built-in address (keep the ?env= and ?src= parameters unless the new site drops them).

Built-in Chromium

An unmanaged browser of the app's own (the one the “with-browser” download ships): org policy can't interfere with its sign-in window, and its version always matches this app. ~170 MB; it lives in the app's data folder and survives updates.
